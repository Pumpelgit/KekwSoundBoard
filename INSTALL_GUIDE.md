# KekwSoundBoard - Install Guide

## 1. Download WowUp
- Go to https://github.com/WowUp/WowUp/releases
- Download the latest `.exe` file and install it
- Open WowUp and make sure it detects your WoW installation

## 2. Install KekwSoundBoard
- In WowUp, click **Get Addons** > **Install from URL**
- Paste this URL:

```
https://github.com/Pumpelgit/KekwSoundBoard/releases/latest/download/KekwSoundBoard.zip
```

- Click Install
- Done! Restart WoW if it's open

## 3. How to use
- Type `/kekw list` in WoW chat to see all available sound emotes
- Type `/kekw play leeroy` to play a sound privately
- Type `/kekw help` for all commands
